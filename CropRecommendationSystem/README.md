# 🌱 AI Crop Recommendation System

An industry-level, end-to-end AI-powered Crop Recommendation System. It uses environmental parameters (temperature, humidity, rainfall) and soil nutrients (Nitrogen, Phosphorus, Potassium, pH) to recommend the optimal crop.

## ✨ Features
- **Machine Learning**: Random Forest model trained for high accuracy. Inference uses exported `.pkl` artifacts.
- **FastAPI Backend**: Robust, validated API using Pydantic schemas. Includes confidence score (`predict_proba()`).
- **OpenWeather API Integration**: Automatically fetches real-time weather if a city is provided. 
- **Modern UI**: A beautiful, responsive frontend built with glassmorphism CSS and vanilla JS.
- **Expert Farming Tips**: Dynamic recommendations for irrigation, soil, and climate for the predicted crop.

## 🚀 Getting Started

### 1. Setup Python Environment
```bash
# Create virtual environment
python -m venv venv2

# Activate it
# On Windows
.\venv2\Scripts\activate
# On Mac/Linux
source venv2/bin/activate

# Install dependencies
pip install -r requirements.txt
```

### 2. Train the Model (Optional)
The pre-trained model is generated by running:
```bash
python train_model.py
```
This downloads the dataset (if missing), trains Decision Tree and Random Forest, and exports the best model into the `model/` directory.

### 3. Configure the Backend
Rename `.env.example` to `.env` (or create one) and add your OpenWeather API key:
```env
OPENWEATHER_API_KEY=your_actual_api_key_here
```
*Note: The system gracefully handles missing weather API keys by falling back to manual temperature/humidity inputs.*

### 4. Run the API Layer
Start the FastAPI server on `localhost:8000`:
```bash
.\venv2\Scripts\python.exe -m uvicorn backend.main:app --reload
```

### 5. Launch the Frontend
Simply open the `frontend/index.html` file in any modern web browser.
Alternatively, serve it using python's built-in HTTP server:
```bash
cd frontend
python -m http.server 8080
```
Then navigate to `http://localhost:8080`.
